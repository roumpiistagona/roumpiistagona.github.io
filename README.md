# roumpiistagona.github.io
Η Ρούμπι είναι μια μικρή, γενναία σταγόνα αίματος με μεγάλη καρδιά. Ταξιδεύει από άνθρωπο σε άνθρωπο, μεταφέροντας αγάπη, ελπίδα και ζωή. Μέσα από το ταξίδι της, μας μαθαίνει πως η αιμοδοσία είναι μια απλή πράξη που μπορεί να κάνει θαύματα. Η Ρούμπι μας θυμίζει πως όλοι μπορούμε να γίνουμε ήρωες. Με μια σταγόνα, με μια καρδιά, με μια προσφορά ζωής!
